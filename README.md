# PLANT-PAL-PYTHON-PROJECT
PROJECT TITLE : PLANT PAL (Reminder to water your plants!)

OVERVIEW
Plant Pal is a simple application that runs in your Python console. Through it, users can input their plants and get a reminder of when to water the plants. 
This project meets the syllabus concepts of computer science problem-solving, modular programming, data structures, and algorithm design in solving a real-world problem.

FEATURES 
- Define plants with watering intervals in days.
- Track last time watered for each plant.
- Water plants by selecting them from a list.
- Review reminders on which plants need watering today.
- Pull a list of all the plants currently entered
- Handle invalid input exceptions and manage errors.

TECHNOLOGIES/TOOLS USED
- Language: Python 3
- Modules: `datetime` (to track date)
- Concepts: Functions, lists, dictionaries, modularization, control flow
- Version Control: Git & GitHub

STEPS TO INSTALL AND RUN
1. Clone the repository: https://github.com/maseeraah/PLANT-PAL-PYTHON-PROJECT-

INSTRUCTIONS FOR TESTING 
1. Add Plant
Choose option 1 from the menu.
Enter plant name (e.g., Tulsi).
Enter watering frequency (e.g., 2 days).
Plant is added successfully.

2. Water Plant
Choose option 2.
Select a plant from the list.
Plant is marked as watered.

3. Check Reminders
Choose option 3.
System shows which plants need watering today.

4. List Plants
Choose option 4.
Displays all plants with their details.
